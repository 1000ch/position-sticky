# position-sticky

## Usage

Load `position-sticky.min.js` and call `PositionSticky()`.

```html
<script src="dist/position-sticky.js"></script>
<script>PositionSticky('#sticky');</script>
```

Top attribute for `#sticky` should be set.

```css
#sticky {
  top: 10px;
}
```
